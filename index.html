AutoAuto

Helping kids owning their pathways to the tech world.

Our team aims to build scalable learning platforms that allow kids to start developing in Python from an early age, gradually leveling them up to more advanced coding methods and languages. Our use cases integrates Python and autonomous vehicles.


US Dept of Labor estimated that in 2020 there will be an estimated 1.4 million computer specialist job openings. However the US university system will only produce enough graduates to fill 29% of these roles...We want to address this problem by building an unique learning plataform which integrates Python and autonomous vehicles.
